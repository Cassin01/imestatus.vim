# imestatus.vim
Reflect IME status and where caps or not on the cursor color.(for mac and ime user)

## Install

### For vim-plug

```vim
Plug 'Cassin/imestatus.vim'
```

## Usage
- caps lock: red

- 半角英数(Roman): steelblue

- Others(ひらがなetc): orange
