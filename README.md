# imestatus.vim
imestatus.vim reflect IME and caps-lock on the cursor color (for mac).

## Install

### For vim-plug
```vim
Plug 'Cassin01/imestatus.vim'
```

## Usage
- caps lock: red

- 半角英数(Roman): steelblue

- Others(ひらがなetc): orange
